Команды Jenkins

  • build — запускает сборку и, при желании, ждёт её завершения
  • cancel-quiet-down — отменяет эффект команды «quiet-down»
  • clear-queue — очищает очередь сборки
  • connect-node — переподключается к узлу
  • console — получает вывод консоли сборки
  • copy-job — копирует задание
  • delete-builds — удаляет записи сборки
  • disable-job — отключает задание
  • login — сохраняет текущие учётные данные, чтобы будущие команды могли выполняться без явной информации о них
  • logout — удаляет учётные данные, сохранённые с помощью команды «login»
  • mail — читает stdin и отправляет его в виде электронной почты
